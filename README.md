# Gulp boilerplate

Gulp starter boilerplate

## Run Locally

Clone the project

```bash
  git clone https://github.com/mbredikhin/gulp-boilerplate.git
```

Go to the project directory

```bash
  cd gulp-boilerplate
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
